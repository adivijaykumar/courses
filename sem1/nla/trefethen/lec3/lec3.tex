\begin{homeworkProblem}
    For something to be a vector norm it should satisfy three conditions. We list down these three conditions and see if $\norm{\vdot}_W = \norm{Wx}$ does indeed satisfy them, 
    \begin{itemize}
    	\item $\norm{x}_W \ge 0$. This is automatically satisfied by the definition of the norm.
    	\item $\norm{x}_W = 0 $ iff $x=0$. As $W$ is a nonsingular matrix by assumption, this property is satisfied as well.
    	\item $\norm{\alpha x}_W = \abs{\alpha} \norm{x}_W$. As $\alpha$ is just a multiplicative constant and $W$ is indeed a matrix, this property is satisfied as well.
    \end{itemize}
    
    Hence, $\norm{\vdot}$ is a vector norm.
\end{homeworkProblem}
